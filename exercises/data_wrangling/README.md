# Module: Data Wrangling

This module focuses on the transformation of raw string data into structured Python objects.

## 🚀 Projects

### [Clean Data](./clean_data/)
A Class-based system for parsing log files. 
* **Objective**: Intake pipe-delimited strings and map them to class attributes for validation and reporting.
* **Files**:
    * `main.py`: Entry point for the application.
    * `log_parser.py`: Contains the `LogParser` class definition.
    * `data.py`: Source list for raw log strings.

## 🛠️ Skills Applied
* Python Class structure and `self` reference.
* String methods (`.split()`, `.strip()`).
* Module-to-module imports.
